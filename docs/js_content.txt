¿Qué  es  Javascript?  
JavaScript  es  un  lenguaje  de  programación  interpretado,  lo  que  significa  que  el

de

JavaScript  es  el  idioma  que  usan  las  páginas  web  para  “hacer  cosas”.   Imagina  que

El  corazón  de  la  web  dinámica  
●  Interactividad :  Botones,  menús,  animaciones,  validaciones  de  formularios.  ●  Manipulación  del  DOM :  Cambiar  el  contenido  y  diseño  de  una  página

 ●  Conexiones  en  tiempo  real :  Como  los  chats,  notificaciones,  o

(gracias

Además,  hoy  JavaScript  es  un  lenguaje  que  funciona  en  cualquier  lado:  
●  Frontend :  Gracias  a  librerías  y  frameworks  como  React,  Vue  o  Angular.  ●  Backend :  Usando  Node.js,  puedes  construir  servidores  rápidos  y  ligeros.  ●  Aplicaciones  multiplataforma :  Con  herramientas  como  Electron  o  React
