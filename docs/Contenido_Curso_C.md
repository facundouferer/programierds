# Curso Completo de Programación en C

## Descripción del Curso

Este curso está diseñado para principiantes que desean aprender programación en C desde cero. Basado en el apunte "Apunte de C para principiantes", este curso cubre todos los aspectos fundamentales del lenguaje C, desde la estructura básica de un programa hasta conceptos avanzados como estructuras dinámicas y manejo de memoria.

## Objetivos del Curso

Al finalizar este curso, serás capaz de:
- Escribir programas en C desde cero
- Comprender y aplicar estructuras de control
- Manejar datos usando arreglos y estructuras
- Crear funciones modulares y reutilizables
- Gestionar memoria dinámicamente
- Desarrollar proyectos completos en C

## Duración

**16 semanas** | **60 horas** de contenido | **Nivel**: Principiante

## Requisitos Previos

- No se requiere experiencia previa en programación
- Conocimientos básicos de matemáticas
- Acceso a un computador con compilador C (recomendado: GCC)

## Metodología de Aprendizaje

1. **Teoría breve**: Conceptos clave de cada unidad
2. **Ejemplo práctico**: Código comentado
3. **Ejercicio guiado**: Paso a paso con ayuda
4. **Ejercicio independiente**: Aplicación de lo aprendido
5. **Proyecto integrador**: Cada unidad con un mini-proyecto

## Certificación

Al completar todas las unidades y el proyecto final, podrás obtener un certificado de finalización que acredita tus conocimientos en programación en C.
