# Planificación Programación I 2025 → Presentaciones C (Programierds)

Documento base analizado: **Planificación de Cátedra Programación I 2025**.

## Mapeo de unidades a páginas Astro

1. **Unidad 1 – Introducción a la Algoritmia y al Lenguaje C**
   - Página: `/presentaciones/introduccion_algoritmia_c`
   - Ejes: proceso, estados, algoritmos, paradigmas y estructura base en C.

2. **Unidad 2 – Estructura de Datos**
   - Página: `/presentaciones/estructuras_datos_c`
   - Ejes: tipos de datos, dominio, operadores, memoria y E/S.

3. **Unidad 3 – Sentencias de Control e Iteración**
   - Página: `/presentaciones/control_iteracion_c`
   - Ejes: decisiones, iteraciones y diseño de flujo.

4. **Unidad 4 – Arreglos, Cadenas y Estructuras Dinámicas**
   - Página: `/presentaciones/arreglos_cadenas_estructuras_dinamicas_c`
   - Ejes: arrays, strings, base de estructuras dinámicas.

5. **Unidad 5 – Modularidad**
   - Página existente: `/presentaciones/modularidad_en_c`
   - Ejes: funciones, procedimientos, scope y parámetros.

6. **Unidad 6 – Programación Orientada a Objetos**
   - Página: `/presentaciones/poo_para_programadores_c`
   - Ejes: puente conceptual estructurado ↔ OO para transición a Java.

## Criterio de diseño

- Se mantiene estética retro-terminal y CRT de `modularidad_en_c`.
- Se conserva navegación como **cards** en `/presentaciones`.
- Todas las nuevas cards se etiquetan en categoría **C** para foco curricular.
